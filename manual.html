Roles & Responsibilities

System Overview

The system operates on a four-tier permission model designed for jurisdictional efficiency.

  • Super Admin: Full access to core settings, site identity, and action management.
  • Admin: Manages users, police stations, and jurisdictional categories.
  • Superior (CP/SP/IC): High-level oversight. Provides instructions via App and resolves via Web.
  • Receptionist: Field entry. Registers new complaints exclusively via Mobile App.

Command Dashboard

Web Portal
  • Jurisdictional Stats: Live counters for Total, Resolved, and Pending complaints.
  • Resolution Tracking: Monitor "Total vs Done" metrics per police station.
  • Critical Alerts: Real-time count of "Sensitive Complain No Action" (High Priority cases > 24 hours).
  • Navigation: Click any station card to view its specific records.

Main Dashboard View Screenshot

Complaint Database

Web Portal
  • Advanced Filtering: Filter by Police Station, Complaint Sub-Category, and custom Date Ranges.
  • Search: Global search across complainant names, phone numbers, and addresses.
  • Reporting: One-click "EXPORT CSV" that respects all active filters.
  • View Modal: Click the Eye Icon to see full complaint details and resolution history.

Resolution & Action Taken

Web Portal Mobile App

STRICT ENTRY SEPARATION:

  • Action Taken (WEB ONLY): Superiors/Admins select resolution types (e.g., FIR Lodged) and enter final details on the Web Portal.
  • Superior Note (APP ONLY): Internal guidance, remarks, or immediate instructions are entered via the Android App.
  • Status View: Both fields are merged in the "Detailed View" for total transparency.

App: Complaint Registration

Mobile App
Receptionist
  • Quick Entry: Designed for rapid data entry during public interaction.
  • Station Locked: Entry automatically assigned to the user's registered police station.
  • Priority Mapping: Categories determine high-priority alerts (High Priority = Immediate Superior Alert).

App: Superior Oversight

Mobile App
Superior (CP/SP/IC)
  • Record Review: Scroll through recent jurisdictional records on the move.
  • Instruction Entry: Exclusive access to "ADD OFFICIAL NOTE" for real-time guidance to the station.
  • One-Touch Call: Direct dial to complainant from the record screen.

Alerts & Notifications

FCM System

NOTIFICATION TYPES:

  • 🚨 Emergency Alert: Sent to Superiors when a "High Priority" complaint is registered.
  • ✅ Action Update: Sent to all stakeholders when a resolution is logged on the Web Portal.
  • 📝 Instruction Alert: Sent when a Superior adds a note via the Mobile App.

Settings & Branding

Super Admin
  • Site Identity: Update Site Name, Logo, Favicon, and Footer text.
  • Action Presets: Define the list of "Action Taken" options used in the resolution dropdown.
  • Storage: All branding assets are securely stored and synced across Web/Mobile views.
